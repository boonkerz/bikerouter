## Review-Notes (4000 Zeichen max)
Wegwiesel ist ein Fahrrad-Routenplaner. Keine Registrierung erforderlich.

Testablauf:
1. App öffnen
2. Auf die Karte tippen, um Punkt A zu setzen
3. An eine andere Stelle tippen, um Punkt B zu setzen → Route wird berechnet
4. Features testen: Höhenprofil unten, farbige Belags-Anzeige, Aktionsleiste unter Stats
5. „Runde"-Tab: Startpunkt setzen, Distanz wählen, „Rundtour berechnen"

Externe Services (alle über HTTPS):
- BRouter-Routing: eigener Server (wegwiesel.app/brouter)
- OpenStreetMap-Overpass: eigener Server (wegwiesel.app/overpass)
- Nominatim (Geocoding): nominatim.openstreetmap.org
- Open-Meteo (Wetter): api.open-meteo.com
- OSM-Tiles: openfreemap.org
