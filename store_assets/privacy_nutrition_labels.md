# App Privacy / Nutrition Labels

In App Store Connect unter „App Privacy" ausfüllen.

## Data Types Collected
**Keine.** Wegwiesel sammelt nichts, was einem Nutzer zuordenbar ist.

## Data Linked to User
**Keine.**

## Data Not Linked to User
**Keine.**

## Third-Party Collection
- OpenStreetMap-Tiles: IP-Adresse kann im Rahmen normaler Webanfragen beim Tile-Anbieter (OpenFreeMap) anfallen, wird aber nicht von uns gespeichert.
- Routing/Overpass: eigener Server, IP-Anfragen werden zur Abuse-Prevention temporär geloggt (siehe Datenschutzerklärung).
- Open-Meteo: Positionsanfragen für Wetter, laut Anbieter ohne Nutzerbindung.
- Nominatim: Adress-Anfragen beim OSM-Service.

## Tracking
**Nein.** Keine Werbe-/Analyse-IDs, kein IDFA, keine Cross-App-Profile.

## Data Use Purposes
- App-Funktionalität (Routing, Geocoding, Wetter, Karten) – anonym, nur zur Durchführung der vom Nutzer initiierten Anfrage.

## Location
- **Approximate Location:** Ja, wenn Nutzer „Mein Standort"-Button drückt.
- **Precise Location:** Ja, wenn Nutzer „Mein Standort"-Button drückt.
- Nicht verknüpft mit Identität, nicht für Tracking genutzt.

## Usage Description Strings (Info.plist)
`NSLocationWhenInUseUsageDescription`:
„Wegwiesel verwendet deinen Standort nur, wenn du ihn auf der Karte anzeigen lässt."
