# WegwieselSync — Connect IQ Store (Deutsch)

## App-Name
WegwieselSync

## Kurzbeschreibung (max 100 Zeichen)
Wegwiesel-Routen direkt vom Handy auf deinen Edge laden — per Bluetooth, ohne Connect-Umweg.

## Beschreibung
WegwieselSync nimmt sechsstellige Codes der Wegwiesel-App an und lädt die zugehörige Route direkt per Bluetooth auf deinen Edge — wie bei Komoot, nur ohne Connect-Sync-Umweg.

So funktioniert's:
1. Im Wegwiesel-Tourenplaner (App oder wegwiesel.app) eine Route bauen und auf „An Garmin senden" tippen.
2. Den sechsstelligen Code, der auf dem Handy erscheint, hier auf dem Edge eingeben.
3. Die Strecke landet als FIT-Course im Streckenmenü deines Edge — sofort fahrbar.

Was Wegwiesel ist: ein schlanker, werbefreier Routenplaner für Fahrrad, Wandern und Auto. Profile für Rennrad, Gravel, Trekking, MTB, Wandern und Auto, alternative Routen, Höhenprofil und farbige Belag-Anzeige. Kein Account, keine Cloud, keine Daten-Sammelei.

Was du brauchst:
• Wegwiesel-App auf dem iPhone oder Android (kostenlos)
• Edge mit aktiver Bluetooth-Verbindung zur Wegwiesel-App
• Diese Connect IQ App auf deinem Edge

Datenschutz:
• Die App spricht ausschließlich mit wegwiesel.app, um die FIT-Datei zum Code abzurufen
• Keine Tracking-Pixel, keine Analytics, keine Werbung
• Codes sind nur 24 Stunden gültig und werden nach Download serverseitig gelöscht

Unterstützte Geräte: Edge 530, 540, 550, 830, 840, 850, 1030, 1030 Plus, 1030 Bontrager, 1040, 1050, MTB, Explore 2.

Wegwiesel und der Sync-Code-Server sind Open-Source-Projekte eines Radfahrers. Feedback bitte an info@thomas-peterson.de.
